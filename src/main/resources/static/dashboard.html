Why is the Meta-Model so much better? This graph highlights the drastic performance improvement obtained by the Meta-Model (Stacking). While baseline algorithms (Logistic Regression, Random Forest, Laplace) rely strictly on single-dimension density cutoffs (such as Failed Logon Counts), the Meta-Model leverages computationally engineered multidimensional features like Round-Trip Time and IP routing behaviors. By pulling the confidence threshold down to 0.1 on these rich features, the Meta-Model avoids False Positives and attains near-perfect Recall, catapulting its F1 Score far ahead of the baseline pack.
